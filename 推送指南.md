## 🎯 快速推送脚本

创建 `push.sh`：

```bash
#!/bin/bash

echo "📤 推送到 GitHub..."
cd /home/admin/openclaw/workspace/paper-formatter

echo "检查状态..."
git status

echo ""
echo "提交历史:"
git log --oneline -5

echo ""
echo "推送中..."
git push origin main --tags

if [ $? -eq 0 ]; then
    echo ""
    echo "✅ 推送成功！"
    echo "查看仓库：https://github.com/taobaoaz/paper-formatter"
else
    echo ""
    echo "❌ 推送失败"
    echo "请检查网络连接和认证信息"
fi
```

使用：
```bash
chmod +x push.sh
./push.sh
```

## 🐛 常见问题

### Q1: 提示 "Permission denied"

**原因：** SSH 密钥未配置

**解决：**
```bash
# 生成密钥
ssh-keygen -t ed25519 -C "taobaoaz@qq.com"

# 添加公钥到 GitHub
cat ~/.ssh/id_ed25519.pub
# 复制到 https://github.com/settings/keys
```

### Q2: 提示 "Authentication failed"

**原因：** 密码错误或 Token 过期

**解决：**
1. 访问 https://github.com/settings/tokens
2. 删除旧 token
3. 生成新 token
4. 重新推送

### Q3: 推送被拒绝

**原因：** 远程仓库有本地没有的提交

**解决：**
```bash
# 先拉取
git pull origin main

# 解决冲突（如果有）

# 再推送
git push origin main --tags
```

### Q4: 大文件推送失败

**原因：** 文件超过 GitHub 限制（100MB）

**解决：**
```bash
# 使用 Git LFS
git lfs install
git lfs track "*.psd"
git add .gitattributes
git add 大文件
git commit -m "添加大文件"
git push origin main
```
