## 🚀 使用方法

### 打开配置

**菜单：**
```
工具 → 格式化配置
```

**快捷键：**
```
Ctrl + Alt + F
```

### 配置流程

```
1. 打开配置对话框
   ↓
2. 切换到各个标签页
   - 字体配置
   - 段落配置
   - 页面配置
   - 标题配置
   ↓
3. 修改参数
   ↓
4. 保存配置
```

## 📝 配置示例

### 典型配置（学位论文）

**字体：**
- 中文：宋体
- 英文：Times New Roman
- 字号：小四（12pt）
- 行距：1.5 倍

**段落：**
- 首行缩进：2 字符
- 段前间距：0 磅
- 段后间距：0 磅

**页面：**
- 纸张：A4（21x29.7cm）
- 上边距：2.54cm
- 下边距：2.54cm
- 左边距：3.17cm
- 右边距：3.17cm

**标题：**
- 一级：三号（16pt）、加粗、"第 1 章"
- 二级：四号（14pt）、加粗、"1.1"
- 三级：小四（12pt）、不加粗、"1.1.1"
