## 🚀 使用方式

### 快速开始

```bash
# 1. 进入目录
cd /home/admin/openclaw/workspace/paper-formatter

# 2. 运行程序
python main_refactored.py

# 或使用一键推送
./push_to_github.sh
```

### 功能使用

#### 封面配置

```
1. 点击工具栏 "📘 封面配置"
2. 配置封面信息
   - 方式 1：手动填写
   - 方式 2：上传模板选择页面
3. 保存配置
4. 格式化时应用
```

#### 模板上传

```
1. 点击 "📂 上传模板并选择页面"
2. 选择 Word 模板 (.docx)
3. 选择封面页（通常第 1 页）
4. 选择声明页（通常第 2 页）
5. 确认保存
```
