## 📦 在 Windows 上打包 .exe

### 方式 1：手动操作（推荐）

#### 步骤 1：准备 Windows 环境

**需要：**
- Windows 10/11 电脑
- Python 3.8+（用于打包，用户不需要）
- 网络连接

#### 步骤 2：克隆仓库

```cmd
git clone https://github.com/taobaoaz/paper-formatter.git
cd paper-formatter
```

**或下载 ZIP：**
1. 访问 https://github.com/taobaoaz/paper-formatter
2. 点击 "Code" → "Download ZIP"
3. 解压到文件夹

#### 步骤 3：安装 Python（仅打包需要）

1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.8+
3. 安装时 ✅ 勾选 "Add Python to PATH"
4. 验证：
   ```cmd
   python --version
   ```

#### 步骤 4：运行打包脚本

```cmd
:: 双击运行
build.bat

:: 或命令行
python build.bat
```

#### 步骤 5：等待打包完成

**时间：** 3-5 分钟

**进度：**
```
[1/4] 检查 PyInstaller...
[2/4] 清理旧文件...
[3/4] 开始打包...
[4/4] 打包完成！
```

#### 步骤 6：获取 .exe 文件

**位置：**
```
dist/论文排版优化器.exe
```

**大小：** 约 50-60MB

---

### 方式 2：使用 GitHub Actions（自动）

#### 步骤 1：启用 GitHub Actions

1. 访问仓库：https://github.com/taobaoaz/paper-formatter
2. 点击 "Actions" 标签
3. 点击 "Enable workflows"

#### 步骤 2：触发打包

1. 点击 "Build Windows EXE" workflow
2. 点击 "Run workflow"
3. 选择分支：main
4. 点击 "Run workflow"

#### 步骤 3：下载产物

**等待 10-15 分钟后：**
1. 点击最新的运行记录
2. 在 "Artifacts" 部分
3. 下载 `论文排版优化器.exe`

## 🐛 常见问题

### Q1: build.bat 运行失败？

**可能原因：**
- Python 未安装
- Python 未添加到 PATH
- 网络问题

**解决：**
```cmd
:: 检查 Python
python --version

:: 重新安装 Python
:: 确保勾选 "Add Python to PATH"

:: 手动安装依赖
pip install pyinstaller
pyinstaller --onefile --windowed --name="论文排版优化器" main.py
```

---

### Q2: 打包后无法运行？

**可能原因：**
- 缺少依赖
- 杀毒软件拦截
- 系统不兼容

**解决：**
1. 检查错误提示
2. 临时关闭杀毒软件
3. 右键 → 以管理员身份运行
4. 查看日志文件

---

### Q3: 功能测试失败？

**处理方法：**
1. 记录错误信息
2. 截图保存
3. 查看日志
4. 提交 Issue
