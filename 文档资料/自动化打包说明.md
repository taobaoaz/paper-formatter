## 🪟 Windows 打包方案

### 方案 A：手动打包（推荐）⭐⭐⭐⭐⭐

**步骤：**

1. **在 Windows 电脑上**
   - 打开 https://github.com/taobaoaz/paper-formatter
   - 下载代码（Clone 或 Download ZIP）

2. **解压后运行打包脚本**
   ```cmd
   :: 双击
   build.bat
   
   :: 或命令行
   python build.bat
   ```

3. **等待 3-5 分钟**

4. **获取 .exe 文件**
   ```
   dist/论文排版优化器.exe
   ```

**优点：**
- ✅ 简单直接
- ✅ 无需配置
- ✅ 立即可用

---

### 方案 B：GitHub Actions（自动）⭐⭐⭐⭐

**需要配置 workflow 权限**

#### 步骤 1：添加 Token 权限

1. 访问：https://github.com/settings/tokens
2. 编辑你的 Token
3. 勾选 ✅ `workflow` 权限
4. 保存

#### 步骤 2：推送工作流文件

```bash
cd /home/admin/openclaw/workspace/paper-formatter
git push origin main
```

#### 步骤 3：触发打包

**方式 1：推送标签**
```bash
git tag v2.0.0
git push origin v2.0.0
```

**方式 2：手动触发**
1. 访问仓库 Actions 标签
2. 选择 "Build Windows EXE"
3. 点击 "Run workflow"

#### 步骤 4：下载产物

**等待 10-15 分钟：**
1. 查看最新的运行记录
2. 在 Artifacts 部分
3. 下载 `论文排版优化器.exe`
