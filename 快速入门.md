## 📦 安装依赖

首次运行前需要安装依赖：

```bash
pip install -r requirements.txt
```

**主要依赖：**
- PyQt5 - GUI 框架
- python-docx - Word 文档处理
- pywinstyles - Windows 样式（可选）

## ✨ 主要功能

### 1. 封面和声明页配置 📘

**位置：** 工具 → 封面和声明页配置

**功能：**
- ✅ 可视化配置封面信息

- ✅ 上传 Word 模板选择页面
- ✅ 配置保存/加载

**快捷键：** Ctrl+Shift+C

**位置：** 封面配置对话框 → 上传模板并选择页面

**功能：**
- ✅ 上传学校 Word 模板
- ✅ 可视化选择封面页
- ✅ 可视化选择声明页
- ✅ 自动提取页面

### 3. 文档格式化 📝

**位置：** 首页 → 快速开始

**功能：**
- ✅ 一键格式化文档
- ✅ 支持多种模板
- ✅ 自动识别格式
- ✅ 批量处理

### 4. 参考文献 📚

**位置：** 首页 → 参考文献

**功能：**
- ✅ 支持 GB/T 7714
- ✅ 支持 APA、MLA、IEEE
- ✅ 自动格式化
- ✅ 批量处理

## 🎯 快速使用

### 场景 1：格式化论文

```
1. 启动程序
2. 点击 "快速开始"
3. 选择输入文件
4. 选择输出路径
5. 选择模板
6. 点击 "开始格式化"
7. 等待完成
```

### 场景 2：配置封面

```
1. 点击 "📘 封面配置"
2. 填写学校名称
3. 填写论文题目等信息
4. 选择声明页类型
5. 点击 "保存配置"
```

### 场景 3：上传模板

```
1. 点击 "封面配置"
2. 点击 "上传模板并选择页面"
3. 选择 Word 模板
4. 选择第 1 页（封面）
5. 选择第 2 页（声明）
6. 点击 "确定"
```

## ❓ 常见问题

### Q1: 启动失败，提示 ModuleNotFoundError

**解决：**
```bash
pip install -r requirements.txt
```

### Q2: 找不到封面配置功能

**解决：**
- 确保运行的是 `run.py` 或 `main_refactored.py`
- 不要运行旧版 `main.py`

### Q3: 界面显示异常

**解决：**
- Windows: 安装 pywinstyles
  ```bash
  pip install pywinstyles
  ```
- Linux/Mac: 使用基础版本（自动）

### Q4: 无法上传模板

**解决：**
- 确保是 .docx 格式
- 检查文件是否损坏
- 重启程序重试
