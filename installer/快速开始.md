**5 分钟快速上手！**

## 📦 方式 1：手动制作安装包

### 步骤 1：安装 Inno Setup

**下载：**
```
https://jrsoftware.org/isdl.php
```

**安装：**
1. 下载安装包
2. 双击运行
3. 完成安装

### 步骤 2：准备图标（可选）

**创建文件：**
```
installer/app_icon.ico
```

**在线生成图标：**
```
https://www.icoconverter.com/
```

### 步骤 3：编译安装包

**方法 A：使用 GUI**

1. 打开 **Inno Setup Compiler**
2. File → Open → 选择 `installer/installer.iss`
3. Build → Compile（或 Ctrl+F9）
4. 完成！

**方法 B：命令行**

```bash
iscc installer/installer.iss
```

### 步骤 4：获取安装包

**位置：**
```
installer_output/论文排版优化器安装程序_v2.0.0.exe
```

## 🔄 测试在线更新

### 方法 1：命令行测试

```bash
cd 功能模块
python auto_updater.py
```

**预期输出：**
```
✅ 当前已是最新版本 (v2.0.0)
```

### 方法 2：在程序中调用

**修改 `核心模块/main.py`：**

```python
# 在菜单栏添加检查更新
def create_menu_bar(self):
    menubar = self.menuBar()
    tools_menu = menubar.addMenu('工具 (&T)')
    
    # 检查更新
    update_action = QAction('🔄 检查更新', self)
    update_action.triggered.connect(self.check_update)
    tools_menu.addAction(update_action)

def check_update(self):
    from auto_updater import check_for_updates
    check_for_updates(self)
```

**运行程序：**
```bash
cd 核心模块
python main.py
```

**点击：** 工具 → 检查更新

## 🐛 常见问题

### Q1: Inno Setup 安装失败？

**A:** 以管理员身份运行安装程序

### Q2: 编译安装包报错？

**A:** 检查路径是否正确，确保 `dist/` 文件夹有 EXE 文件

### Q3: GitHub Actions 失败？

**A:** 检查 `.github/workflows/build-installer.yml` 配置

### Q4: 更新检查失败？

**A:** 检查网络连接，确保能访问 GitHub
