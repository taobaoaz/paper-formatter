**文件：** `installer/installer.iss`

**功能：**
- ✅ 专业安装向导
- ✅ 自定义安装目录
- ✅ 创建快捷方式
- ✅ 自动卸载功能
- ✅ 安装完成运行程序

**制作安装包步骤：**

#### 方式 1：使用 Inno Setup（推荐）

**步骤：**

1. **下载 Inno Setup**
   ```
   https://jrsoftware.org/isdl.php
   ```

2. **安装 Inno Setup**
   - 双击安装
   - 按提示完成

3. **打开脚本**
   - 启动 Inno Setup Compiler
   - File → Open → 选择 `installer/installer.iss`

4. **编译安装包**
   - Build → Compile
   - 或按 Ctrl + F9

5. **获取安装包**
   ```
   installer_output/论文排版优化器安装程序_v2.0.0.exe
   ```

#### 方式 2：命令行编译

```bash
iscc installer/installer.iss
```

## 🔧 集成到主程序

### 1. 添加更新菜单

**修改文件：** `核心模块/main.py`

**在 `create_menu_bar` 方法中添加：**

```python
def create_menu_bar(self):
    menubar = self.menuBar()
    
    # 工具菜单
    tools_menu = menubar.addMenu('工具 (&T)')
    
    # ... 其他菜单项 ...
    
    # 检查更新
    update_action = QAction('🔄 检查更新', self)
    update_action.setShortcut('Ctrl+U')
    update_action.triggered.connect(self.check_update)
    tools_menu.addAction(update_action)
```

### 2. 添加更新方法

**在 `MainWindow` 类中添加：**

```python
def check_update(self):
    """检查更新"""
    from auto_updater import check_for_updates
    check_for_updates(self)
```

### 3. 启动时检查更新

**在 `__init__` 方法中添加：**

```python
def __init__(self):
    super().__init__()
    # ... 其他初始化代码 ...
    
    # 延迟检查更新（5 秒后）
    from PyQt5.QtCore import QTimer
    QTimer.singleShot(5000, self.auto_check_update)

def auto_check_update(self):
    """自动检查更新"""
    from auto_updater import AutoUpdater
    
    updater = AutoUpdater()
    update_info = updater.check_update()
    
    if update_info.get('has_update'):
        # 显示更新提示（不强制）
        self.tray_icon.showMessage(
            '发现新版本',
            f'发现新版本 {update_info["latest_version"]}，点击"工具"菜单检查更新',
            QSystemTrayIcon.Information,
            3000
        )
```
