## 📋 制作安装包

### 方式 1：使用脚本（推荐）

#### 步骤 1：准备文件

**确保以下文件存在：**
```
paper-formatter/
├── dist/
│   └── 论文排版优化器.exe  (打包好的程序)
├── installer/
│   └── installer.iss       (安装脚本)
└── 功能模块/
    └── auto_updater.py     (自动更新模块)
```

#### 步骤 2：打开脚本

1. 启动 **Inno Setup Compiler**
2. 点击 **File → Open**
3. 选择 `installer/installer.iss`

#### 步骤 3：编译安装包

1. 点击 **Build → Compile**
2. 或按 **Ctrl + F9**
3. 等待编译完成

#### 步骤 4：获取安装包

**安装包位置：**
```
installer/论文排版优化器安装程序_v2.0.0.exe
```

**文件大小：** 约 60-70MB

## ⚙️ 高级配置

### 自定义图标

**准备图标文件：**
```
installer/app_icon.ico
```

**图标要求：**
- 格式：.ico
- 尺寸：256x256（推荐）
- 包含多个尺寸（16, 32, 48, 256）

**修改脚本：**
```iss
SetupIconFile=installer\app_icon.ico
UninstallDisplayIcon={app}\论文排版优化器.exe
```

### 自定义安装页面

**添加欢迎页面：**
```iss
[Code]
procedure InitializeWizard;
begin
  WizardForm.WelcomeLabel1.Caption := '欢迎安装 论文排版优化器 v2.0.0';
  WizardForm.WelcomeLabel2.Caption := '此向导将引导您完成安装过程...';
end;
```

### 添加运行库

**如果需要包含 Python：**

```iss
[Files]
Source: "python-3.10.exe"; DestDir: "{tmp}"; Flags: deleteafterinstall

[Run]
Filename: "{tmp}\python-3.10.exe"; Parameters: "/quiet InstallAllUsers=1"; StatusMsg: "安装 Python..."
```

### 注册表设置

**添加文件关联：**

```iss
[Registry]
Root: HKCR; Subkey: ".docx"; ValueType: string; ValueData: "PaperFormatter.Document"; Flags: uninsdeletevalue
Root: HKCR; Subkey: "PaperFormatter.Document"; ValueType: string; ValueData: "论文排版优化器文档"
```

## 🚀 自动更新功能

### 集成更新模块

#### 在主程序中添加

**修改 `main.py`：**

```python
# 在菜单栏添加检查更新
def check_update(self):
    from auto_updater import check_for_updates
    check_for_updates(self)

# 添加到菜单
tools_menu = menubar.addMenu('工具 (&T)')
update_action = QAction('🔄 检查更新', self)
update_action.triggered.connect(self.check_update)
tools_menu.addAction(update_action)
```

#### 启动时检查

```python
# 在程序启动时检查更新
def __init__(self):
    super().__init__()
    # ... 其他初始化代码
    
    # 延迟检查更新
    QTimer.singleShot(5000, self.check_update_on_startup)

def check_update_on_startup(self):
    """启动时检查更新"""
    from auto_updater import AutoUpdater
    
    updater = AutoUpdater()
    update_info = updater.check_update()
    
    if update_info.get('has_update'):
        # 提示用户更新
        self.show_update_notification(update_info)
```

## 📊 GitHub Actions 自动打包

### 创建工作流

**文件：** `.github/workflows/build-installer.yml`

```yaml
name: Build Installer

on:
  push:
    tags:
      - 'v*'

jobs:
  build-installer:
    runs-on: windows-latest
    
    steps:
    - uses: actions/checkout@v4
    
    - name: Set up Python
      uses: actions/setup-python@v5
      with:
        python-version: '3.10'
    
    - name: Install dependencies
      run: |
        pip install pyinstaller
        pip install -r 启动文件/requirements.txt
    
    - name: Build EXE
      run: |
        pyinstaller --onefile --windowed --name="论文排版优化器" main.py
    
    - name: Compile Installer
      uses: minhcasi/iscc-action@v1
      with:
        script-path: installer/installer.iss
    
    - name: Upload Installer
      uses: actions/upload-artifact@v4
      with:
        name: 论文排版优化器安装程序
        path: installer/论文排版优化器安装程序_*.exe
    
    - name: Create Release
      uses: softprops/action-gh-release@v1
      with:
        files: installer/论文排版优化器安装程序_*.exe
      env:
        GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```
