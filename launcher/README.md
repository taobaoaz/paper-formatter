# 🚀 启动说明

**论文排版优化器 v2.0**

---

## 📋 快速启动

### Windows 用户

**双击运行：**
```
启动.bat
```

### Linux/Mac 用户

**终端运行：**
```bash
chmod +x 启动.sh
./启动.sh
```

### 命令行启动

```bash
python launcher.py
```

---

## 📁 文件说明

| 文件 | 说明 |
|------|------|
| `启动.bat` | Windows 一键启动 |
| `启动.sh` | Linux/Mac 启动脚本 |
| `launcher.py` | Python 启动器 |
| `requirements.txt` | Python 依赖列表 |
| `安装说明.md` | 详细安装指南 |

---

## ⚙️ 安装依赖

**首次运行前：**

```bash
pip install -r requirements.txt
```

**主要依赖：**
- PyQt5 - GUI 框架
- python-docx - Word 处理
