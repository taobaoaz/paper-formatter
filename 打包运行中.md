### Step 1: Checkout ✅

```yaml

```

**状态：** 已完成  
**作用：** 下载仓库代码

---

### Step 2: Set up Python ⏳

```yaml
- uses: actions/setup-python@v4
  with:
    python-version: '3.10'
```

**状态：** 进行中/已完成  
**作用：** 安装 Python 3.10

---

### Step 3: Install dependencies ⏳

```yaml
- run: |
    python -m pip install --upgrade pip
    pip install pyinstaller
    pip install -r requirements.txt
```

**状态：** 等待中  
**作用：** 安装 PyInstaller 和项目依赖

**预计时间：** 2-3 分钟

---

### Step 4: Build EXE ⏳

```yaml
- run: |
    pyinstaller --onefile --windowed \
      --name="论文排版优化器" \
      --add-data "docs;docs" \
      --hidden-import PyQt5 \
      --hidden-import python-docx \
      main.py
```

**状态：** 等待中  
**作用：** 打包生成 .exe 文件

**预计时间：** 5-8 分钟

---

### Step 5: Upload artifact ⏳

```yaml
- uses: actions/upload-artifact@v3
  with:
    name: 论文排版优化器-Windows
    path: dist/论文排版优化器.exe
```

**状态：** 等待中  
**作用：** 上传 .exe 文件到 GitHub

**预计时间：** 1-2 分钟
