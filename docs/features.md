# 功能详解

本文档详细介绍 Paper Formatter 的所有功能和特性。

## 📘 封面和声明页配置

### 可视化配置界面
- **模板上传**：支持上传 Word 模板文件
- **封面选择**：从模板库选择封面样式
- **声明页配置**：配置声明页内容和格式
- **实时预览**：配置时实时预览效果

### 支持的模板格式
- 学术论文封面
- 学位论文封面
- 期刊投稿封面
- 会议论文封面
- 自定义封面模板

### 配置保存与加载
- **保存配置**：将配置保存为 `.json` 文件
- **加载配置**：从文件加载已有配置
- **配置模板**：创建和管理配置模板
- **批量应用**：将配置应用到多个文档

## 📄 模板页面选择

### 预置模板库
- **学术论文模板**：标准学术格式
- **期刊投稿模板**：常见期刊格式要求
- **学位论文模板**：硕士/博士论文格式
- **会议论文模板**：会议投稿格式
- **报告模板**：技术报告格式

### 自定义模板
- **模板编辑器**：可视化模板编辑
- **样式定义**：定义字体、段落、页边距等
- **变量支持**：支持动态变量（如标题、作者、日期）
- **模板导入/导出**：分享和重用模板

### 模板管理
- **分类管理**：按类型分类模板
- **搜索功能**：快速查找模板
- **收藏功能**：标记常用模板
- **模板更新**：在线更新模板库

## 📝 文档格式化

### 一键格式化
- **智能识别**：自动识别文档结构
- **批量处理**：同时处理多个文档
- **进度显示**：实时显示处理进度
- **错误处理**：自动处理格式错误

### 支持的文档格式
- **Microsoft Word**：.docx, .doc
- **纯文本**：.txt
- **Markdown**：.md
- **其他格式**：通过插件支持

### 格式化选项
- **页面设置**：纸张大小、页边距、方向
- **字体设置**：中英文字体、字号、颜色
- **段落设置**：行距、缩进、对齐方式
- **标题样式**：多级标题样式
- **页眉页脚**：自定义页眉页脚内容
- **页码格式**：多种页码样式

## 📚 参考文献管理

### 引用样式支持
- **APA 格式**：心理学、教育学等
- **MLA 格式**：人文学科
- **Chicago 格式**：历史、艺术等
- **IEEE 格式**：工程、计算机科学
- **GB/T 7714**：中国国家标准
- **自定义样式**：创建自定义引用格式

### 参考文献库
- **本地库**：管理个人参考文献
- **在线搜索**：从学术数据库搜索
- **导入导出**：支持 BibTeX, EndNote, Zotero
- **自动更新**：定期更新引用信息

### 引用工具
- **引用插入**：在文档中插入引用
- **文献列表**：自动生成参考文献列表
- **格式检查**：检查引用格式正确性
- **重复检测**：检测重复引用

## 🤖 AI 智能识别

### 文档结构识别
- **章节识别**：自动识别文档章节结构
- **标题识别**：识别多级标题
- **图表识别**：识别图表和题注
- **公式识别**：识别数学公式

### 智能建议
- **格式建议**：基于内容推荐格式
- **样式优化**：优化文档样式
- **错误检测**：检测格式错误
- **改进建议**：提供改进建议

### 学习功能
- **用户习惯学习**：学习用户的格式偏好
- **模板推荐**：基于历史使用推荐模板
- **智能调整**：自动调整格式参数

## 🔤 字体管理（v2.2.2+）

### 在线字体搜索
- **字体库**：访问 Google Fonts 等字体库
- **中文优先**：优先显示中文字体
- **实时预览**：预览字体效果
- **字体筛选**：按分类、风格筛选

### 字体下载与安装
- **多线程下载**：快速下载字体包
- **进度显示**：实时显示下载进度
- **自动安装**：下载后自动安装
- **字体管理**：管理已安装字体

### 字体应用
- **一键应用**：将字体应用到文档
- **字体组合**：推荐字体组合
- **字体替换**：批量替换文档字体
- **字体备份**：备份字体配置

## 📦 自动备份管理（v2.1.8+）

- **触发条件**：保存前、格式化前自动备份

- **压缩选项**：启用/禁用压缩
