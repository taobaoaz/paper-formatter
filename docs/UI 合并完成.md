## ✨ 核心功能

### 封面和声明页配置

**位置：** 工具栏 → 📘 封面配置

**功能：**
- ✅ 可视化配置界面
- ✅ 支持 4 种声明页类型
- ✅ 上传 Word 模板选择页面
- ✅ 配置保存/加载

**使用方式：**
```
1. 点击"📘 封面配置"
2. 填写封面信息
3. 或上传模板选择页面
4. 保存配置
5. 格式化时自动应用
```

### 模板页面选择

**位置：** 封面配置对话框 → 上传模板并选择页面

**功能：**
- ✅ 上传 Word 模板
- ✅ 可视化选择页面
- ✅ 自动提取封面和声明页
- ✅ 保留原模板格式
