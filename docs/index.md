# Paper Formatter - 文档

欢迎来到 Paper Formatter 文档网站！这是一个专业的学术论文格式化工具，帮助研究人员、学生和作者快速格式化论文文档。

## ✨ 主要功能

- **批量处理**：支持同时处理多个文档
- **自动备份**：格式化前自动备份原文件
- **智能清理**：自动管理备份文件
- **多格式支持**：支持多种文档格式
- **用户友好界面**：简洁直观的操作界面

## 🚀 快速开始

### 安装

1. **下载最新版本**：[GitHub Releases](https://github.com/taobaoaz/paper-formatter/releases)
2. **运行安装程序**：双击安装包并按照提示操作
3. **启动应用**：从开始菜单或桌面快捷方式启动

### 基本使用

1. **打开文档**：点击"打开"按钮选择文档
2. **选择格式**：从预设格式中选择或自定义
3. **开始格式化**：点击"格式化"按钮
4. **保存结果**：格式化完成后保存文档
