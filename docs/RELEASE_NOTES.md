## 🎉 重大更新

### 新增功能

#### 1. 封面和声明页配置器 📘

**全新的可视化配置界面**

- ✅ 支持手动配置封面信息
  - 学校名称
  - 论文类型（本科/硕士/博士/课程）
  - 6 个封面字段（论文题目、作者、学号等）
  
- ✅ 支持 4 种声明页类型
  - 独创性声明
  - 版权使用授权书
  - 独创性声明 + 授权书
  - 保密论文声明

- ✅ 自定义声明内容
  - 支持输入自定义声明文本
  - 适应特殊学校要求

- ✅ 配置保存/加载
  - 保存为 JSON 配置文件
  - 下次使用时快速加载

#### 2. 模板页面选择器 📄

**上传学校模板，自动提取封面和声明页**

- ✅ 上传 Word 模板文件 (.docx)

- ✅ 自动提取指定页面
- ✅ 完美保留原模板格式

  - 特殊排版

#### 3. 现代化 UI 设计 🎨

**Material Design + Fluent UI 风格**

- ✅ 响应式交互

## 🚀 使用指南

### 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/taobaoaz/paper-formatter.git
cd paper-formatter

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行程序
python main_refactored.py
```

### 使用封面配置器

```
1. 点击工具栏 "📘 封面配置"
2. 配置封面信息
   - 方式 1：手动填写
   - 方式 2：上传模板选择页面
3. 点击 "💾 保存配置"
4. 格式化时自动应用
```

### 上传学校模板

```
1. 点击 "📂 上传模板并选择页面"
2. 选择 Word 模板文件 (.docx)
3. 选择封面页（通常第 1 页）
4. 选择声明页（通常第 2 页）
5. 点击 "✅ 确定"
```
