## 🚀 使用方式

### 方式 1：直接运行

```bash
cd paper-formatter
python main_refactored.py
```

### 方式 2：使用更新脚本

**Windows:**
```cmd
update.bat
python main_refactored.py
```

**Linux/Mac:**
```bash
./update.sh
python3 main_refactored.py
```

### 方式 3：从 GitHub 克隆

```bash
git clone https://github.com/taobaoaz/paper-formatter.git
cd paper-formatter
pip install -r requirements.txt
python main_refactored.py
```
