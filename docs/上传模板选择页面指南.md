# 📄 上传模板并选择页面指南

**让软件完全符合你学校的格式要求！**

---

## 🎯 功能说明

这个功能允许你：
1. **上传学校的 Word 模板**
2. **手动选择哪两页是封面和声明页**
3. **软件自动提取并应用**

**适用场景：**
- ✅ 学校提供了 Word 模板文件
- ✅ 模板有特殊的格式要求
- ✅ 包含校徽、特殊表格等元素
- ✅ 手动配置无法满足要求

---

## 🚀 使用步骤

### 步骤 1：打开配置器

在主界面点击：
```
工具 → 封面和声明页配置
```

### 步骤 2：上传模板

点击 **"📂 上传模板并选择页面"** 按钮：

```
┌─────────────────────────────────────┐
│  📄 上传学校模板（可选）            │
├─────────────────────────────────────┤
│  如果学校提供了 Word 模板，可以上传  │
│  后选择封面和声明页                 │
│  软件会自动提取并应用到你的论文中   │
│                                     │
│  [📂 上传模板并选择页面]            │
│  未上传模板                         │
└─────────────────────────────────────┘
```

### 步骤 3：选择 Word 模板文件

在弹出的文件选择器中：
```
1. 找到学校的 Word 模板文件
2. 选择文件（.docx 格式）
3. 点击 "打开"
```

**支持的文件格式：**
- ✅ `.docx` - Word 2007+ 文档

### 步骤 4：选择封面和声明页

上传后会自动打开页面选择器：


## 💡 页面选择技巧

### 如何确定哪页是封面？

**封面特征：**
- ✅ 包含学校名称
- ✅ 包含论文题目
- ✅ 包含作者信息
- ✅ 通常是第 1 页

### 如何确定哪页是声明页？

**声明页特征：**
- ✅ 标题是"独创性声明"或"声明"
- ✅ 包含声明正文
- ✅ 底部有签名区域
- ✅ 通常是第 2 页

### 如果模板有多页怎么办？

**场景：** 模板包含封面、声明、摘要等多页

**解决方法：**
```
1. 只选择封面和声明页
2. 其他页面（摘要、目录等）不需要选择
3. 软件会自动提取选中的两页
```

### 如果封面和声明在同一页怎么办？

**场景：** 某些学校的模板把封面和声明放在同一页

**解决方法：**
```
1. 选择该页作为封面
2. 声明页选择"无"或跳过
3. 在自定义声明内容中填写
```

## 📋 配置文件格式

保存的配置文件会包含模板信息：

```json
{
  "template": {
    "enabled": true,
    "template_path": "C:/Users/XX/Downloads/清华大学论文模板.docx",
    "page_selection": {
      "cover_page_index": 0,
      "declaration_page_index": 1,
      "template_path": "C:/Users/XX/Downloads/清华大学论文模板.docx"
    }
  },
  "cover": {
    "enabled": true,
    "type": "from_template",
    "school_name": "清华大学",
    ...
  },
  "declaration": {
    ...
  }
}
```

**字段说明：**
- `template.enabled`: 是否使用模板
- `template.template_path`: 模板文件路径
- `template.page_selection`: 页面选择信息
  - `cover_page_index`: 封面页索引（从 0 开始）
  - `declaration_page_index`: 声明页索引（从 0 开始）

## ❓ 常见问题

### Q1: 上传后显示页数不对，怎么办？

**可能原因：**
- Word 文档的实际页数与显示不符
- 分节符导致页数计算错误

**解决方法：**
```
方法 1：手动检查
  - 用 Word 打开模板
  - 数一下实际页数
  - 选择正确的页码

方法 2：预览确认
  - 在页面列表中查看
  - 点击页面查看内容
  - 确认是封面/声明页
```

### Q2: 选择页面后提示错误，怎么办？

**可能原因：**
- 模板文件损坏
- 文件格式不支持

**解决方法：**
```
1. 用 Word 打开模板确认
2. 另存为新的 .docx 文件
3. 重新上传
```

### Q3: 可以只上传封面，不上传声明页吗？

**回答：** 可以。

**操作：**
```
1. 上传模板
2. 只选择封面页
3. 声明页选择"无"或跳过
4. 在配置器中手动配置声明页
```

### Q4: 模板文件太大（>10MB），怎么办？

**回答：** 可以处理，但可能较慢。

**建议：**
```
1. 用 Word 打开模板
2. 删除不必要的图片
3. 压缩图片
4. 另存为新文件
5. 上传处理后的文件
```

### Q5: 上传的模板包含目录和正文，怎么办？

**回答：** 没问题，只选择需要的页面。

**操作：**
```
1. 上传完整模板
2. 只选择封面页（第 1 页）
3. 只选择声明页（第 2 页）
4. 其他页面会被忽略
```
