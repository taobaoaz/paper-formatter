## 🎯 用户使用方式

### Windows 用户

1. 克隆仓库
   ```bash
   git clone https://github.com/taobaoaz/paper-formatter.git
   cd paper-formatter
   ```

2. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```

3. 启动程序
   ```
   双击 启动.bat
   ```

### Linux/Mac 用户

1. 克隆仓库
   ```bash
   git clone https://github.com/taobaoaz/paper-formatter.git
   cd paper-formatter
   ```

2. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```

3. 启动程序
   ```bash
   ./启动.sh
   ```
