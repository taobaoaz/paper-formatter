## 🔄 数据流

### 格式化流程

```
用户操作
   ↓
选择输入文件 (.docx)
   ↓
选择模板 (从模板列表)
   ↓
配置封面和声明页 (可选)
   ↓
点击"开始格式化"
   ↓
┌─────────────────────────────────────┐
│ WorkerThread (后台线程)             │
├─────────────────────────────────────┤
│ 1. 加载输入文档                      │
│ 2. 应用模板格式                      │
│ 3. 添加封面和声明页                  │
│ 4. 保存输出文档                      │
└─────────────────────────────────────┘
   ↓
更新进度条
   ↓
显示完成消息
   ↓
输出文件
```

### 示例 1：简单格式化

```python
from main_refactored import PaperFormatterMainWindow
```

## 📊 性能优化

### 1. 多线程处理

**问题：** 大文档格式化时 UI 卡顿

**解决：** 使用 WorkerThread 后台处理

```python
class WorkerThread(QThread):
    def run(self):
        # 后台执行耗时任务
        result = self.task_func(*self.args, **self.kwargs)
        self.finished.emit(result)
```

### 2. 懒加载

**问题：** 启动时加载所有模板慢

**解决：** 按需加载

```python
def load_template(self, index):
    # 只在需要时加载
    if not self.templates_loaded:
        self.template_manager.load_all()
        self.templates_loaded = True
```

### 3. 缓存机制

**问题：** 重复解析相同文档

**解决：** 使用缓存

```python
from functools import lru_cache

@lru_cache(maxsize=100)
def parse_document(path):
    return DocumentParserFactory.parse(path)
```
