## 🎯 使用指南

### 打开模板管理

**方法 1：菜单**
```
工具 → 模板管理
```

**方法 2：快捷键**
```
Ctrl + Alt + T
```

### 创建模板

**步骤：**
1. 打开模板管理
2. 点击"➕ 新建模板"
3. 填写模板信息
4. 点击"💾 保存模板"

### 导入模板

**步骤：**
1. 打开模板管理
2. 点击"📥 导入模板"
3. 选择模板文件（.json）
4. 模板自动加载

### 导出模板

**步骤：**
1. 选择要导出的模板
2. 点击"📤 导出模板"
3. 选择保存位置
4. 保存为 .json 文件
