## 🎯 使用指南

### 自动备份设置

**打开方式：**
- 菜单：设置 → 🔄 自动备份设置
- 快捷键：Ctrl+Alt+B
- 状态栏：点击备份状态标签

**配置步骤：**
1. 勾选"启用自动备份"
2. 选择备份间隔（建议 10-15 分钟）
3. 设置清理策略（保留 20 个/24 小时）
4. 勾选"保护重要快照"
5. 点击"保存"

**状态栏显示：**
- 🔄 自动备份：10 分钟后 - 正常状态
- 🔄 自动备份：已启用 - 等待备份
- ⏸️ 自动备份：已禁用 - 已禁用

---

### 重要性标记

**标记重要快照：**
1. 打开文档快照管理（Ctrl+Alt+S）
2. 选择要标记的快照
3. 点击"⭐ 标记为重要"
4. 快照列表中高亮显示（黄色背景）

**筛选重要快照：**
1. 打开文档快照管理
2. 勾选"🔍 只看重要"
3. 只显示重要快照
4. 再次勾选取消筛选

**取消重要标记：**
1. 选择已标记的快照
2. 点击"🌟 取消重要标记"
3. 快照恢复正常状态

## 📝 使用场景

### 场景 1：长时间编辑自动备份

```
1. 打开文档开始编辑
2. Ctrl+Alt+B 打开自动备份设置
3. 启用自动备份（10 分钟间隔）
4. 状态栏显示"🔄 自动备份：10 分钟后"
5. 每 10 分钟自动创建快照
6. 无需手动备份，数据安全
```

### 场景 2：标记重要版本

```
1. 完成重要修改
2. Ctrl+Alt+S 打开快照管理
3. 选择最新快照
4. 点击"⭐ 标记为重要"
5. 快照黄色高亮显示
6. 不会被自动清理
```

### 场景 3：查看重要快照

```
1. 打开快照管理
2. 勾选"🔍 只看重要"
3. 只显示重要快照
4. 快速找到关键版本
5. 取消勾选显示全部
```

### 场景 4：调整备份策略

```
1. 频繁编辑阶段：设置 5 分钟间隔
2. 正常编辑阶段：设置 10-15 分钟间隔
3. 偶尔编辑阶段：设置 30-60 分钟间隔
4. 外出前：禁用自动备份
```
