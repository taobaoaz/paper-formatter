## 🎯 使用指南

### 自动快照

**打开文档时：**
```
1. 点击"打开文件"
2. 选择文档
3. 状态栏显示"已自动创建文档快照"（2 秒）
4. 快照数量自动更新
```

**格式化文档时：**
```
1. 选择模板
2. 点击"格式化文档"
3. 状态栏显示"已创建格式化前快照"（2 秒）
4. 快照数量自动更新
5. 执行格式化
```

### 状态栏快照数量

**查看快照数量：**
- 状态栏右侧永久显示：📸 X 个快照

**快速访问快照管理：**
- 点击快照数量标签
- 打开文档快照管理对话框

**更新时间：**
- 打开文档后
- 格式化文档后
- 创建/删除快照后

## 📝 使用场景

### 场景 1：安全格式化

```
1. 打开文档（自动创建"打开文档"快照）
2. 选择模板
3. 点击格式化（自动创建"格式化前"快照）
4. 查看结果
5. 如不满意，Ctrl+Alt+S 打开快照管理
6. 恢复到格式化前的快照
```

### 场景 2：多文档编辑

```
1. 打开文档 A（自动创建快照）
2. 编辑、格式化
3. 打开文档 B（自动创建快照）
4. 编辑、格式化
5. 需要时分别恢复任一快照
```

### 场景 3：快速访问快照

```
1. 编辑过程中
2. 瞥见状态栏"📸 5 个快照"
3. 点击快照数量
4. 查看和管理所有快照
5. 关闭对话框继续编辑
```
