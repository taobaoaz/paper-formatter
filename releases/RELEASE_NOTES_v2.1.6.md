## 🎯 使用指南

### 文档快照管理

**打开方式：**
- 菜单：编辑 → 文档快照管理
- 快捷键：Ctrl+Alt+S

**创建快照：**
1. 点击「➕ 创建快照」按钮
2. 选择文档文件（如未打开）
3. 输入描述（可选）
4. 确认创建

**恢复快照：**
1. 在列表中选择快照
2. 点击「↩️ 恢复到此快照」按钮
3. 确认恢复操作

**搜索快照：**
- 在搜索框输入关键词（支持文件名、描述）
- 使用类型过滤器（全部/完整/增量）

---

### 配置快照管理

**打开方式：**
- 菜单：工具 → 配置快照管理
- 快捷键：Ctrl+Alt+K

**创建快照：**
1. 点击「➕ 创建快照」按钮
2. 输入名称、描述、标签
3. 或点击「📋 从当前配置创建」快速保存当前配置

**恢复配置：**
1. 在列表中选择快照
2. 点击「↩️ 恢复此配置」按钮
3. 确认恢复操作

**比较差异：**
1. 选择一个快照
2. 点击「📊 比较差异」按钮
3. 选择另一个快照进行比较
4. 查看配置差异

**导入/导出：**
- 导出：选择快照 → 点击「📤 导出配置」→ 选择保存位置
- 导入：点击「📥 导入配置」→ 选择 JSON 文件

## 📝 使用场景

### 场景 1：格式化前备份

```
1. 打开文档
2. Ctrl+Alt+S 打开文档快照管理
3. 创建快照（描述：格式化前）
4. 执行格式化
5. 如果不满意，恢复快照
```

### 场景 2：多配置管理

```
1. 配置好学校标准参数
2. Ctrl+Alt+K 打开配置快照管理
3. 从当前配置创建快照（名称：学校标准，标签：学校）
4. 配置好企业文档参数
5. 创建快照（名称：企业配置，标签：企业）
6. 需要时快速切换
```

### 场景 3：配置分享

```
1. 导出配置快照为 JSON
2. 发送给同学/同事
3. 对方导入配置
4. 使用相同标准
```

### 场景 4：配置比较

```
1. 选择当前使用的配置
2. 点击「比较差异」
3. 选择另一个配置
4. 查看参数差异，决定是否切换
```
