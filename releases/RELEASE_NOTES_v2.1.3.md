- **智能识别：** 自动识别标题级别

- **标题统计：** 各级标题数量

## 🎯 使用指南

### 打开文件预览

**方法 1：菜单**
```
工具 → 文件预览
```

**方法 2：快捷键**
```
Ctrl + Alt + P
```

### 预览文件

**步骤：**
1. 打开文件预览对话框
2. 点击"📂 打开文件"
3. 选择 Word 文档
4. 查看预览内容

### 查看结构

**功能：**
- 左侧显示文档结构树
- 右侧显示预览内容
- 可拖动分割线调整比例

### 查看统计

**位置：** 底部状态栏

**显示：**
```
字数：12345 | 段落：100 | 标题：15
```

## 📝 识别规则

### 标题识别

**一级标题：**
- 样式包含"Heading 1"或"标题 1"
- 文本匹配"第 X 章"
- 文本匹配数字开头（如"1、"）

**二级标题：**
- 样式包含"Heading 2"或"标题 2"
- 文本匹配"1.1"格式

**三级标题：**
- 样式包含"Heading 3"或"标题 3"
- 文本匹配"1.1.1"格式
