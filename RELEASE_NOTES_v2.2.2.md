## 🎯 使用指南

### 字体搜索下载

**搜索字体：**
1. 点击"帮助" → "字体管理"
2. 或使用快捷键 Ctrl+Alt+F
3. 切换到"搜索下载"标签页
4. 输入字体名称（可选）
5. 勾选"只搜索中文字体"
6. 点击"搜索"

**下载字体：**
1. 在搜索结果中选择字体
2. 点击"下载选中"
3. 等待下载完成
4. 选择是否立即安装

### 字体管理

**导入字体：**
1. 打开字体管理对话框
2. 点击"导入字体"
3. 选择字体文件（支持多选）
4. 自动安装到系统

**导出字体：**
1. 在字体列表中选择字体
2. 点击"导出字体"
3. 选择保存目录
4. 字体文件复制到目录

**卸载字体：**
1. 在字体列表中选择字体
2. 点击"卸载选中"
3. 确认卸载
4. 字体从系统移除

### API 设置

**获取 Google Fonts API 密钥：**
1. 访问 https://developers.google.com/fonts/docs/developer_api
2. 创建 Google Cloud 项目
3. 启用 Fonts Developer API
4. 创建 API 密钥
5. 复制到字体管理的 API 设置

**保存 API 密钥：**
1. 打开字体管理对话框
2. 切换到"API 设置"标签页
3. 粘贴 API 密钥
4. 点击"保存设置"

## 📝 使用场景

### 场景 1：下载中文字体

```
1. 打开字体管理（Ctrl+Alt+F）
2. 切换到"搜索下载"
3. 勾选"只搜索中文字体"
4. 点击"搜索"
5. 选择"思源黑体"
6. 点击"下载选中"
7. 下载完成后选择"是"安装
8. 字体立即可用
```

### 场景 2：批量导入字体

```
1. 下载多个字体文件
2. 打开字体管理
3. 点击"导入字体"
4. 选择所有字体文件
5. 自动安装到系统
6. 刷新列表查看
```

### 场景 3：导出常用字体

```
1. 打开字体管理
2. 在已安装列表中选择常用字体
3. 点击"导出字体"
4. 选择保存目录（如 U 盘）
5. 在新电脑上导入
6. 保持字体一致性
```

### 场景 4：清理不需要的字体

```
1. 打开字体管理
2. 查看已安装字体列表
3. 选择不需要的字体
4. 点击"卸载选中"
5. 确认卸载
6. 释放系统空间
```
