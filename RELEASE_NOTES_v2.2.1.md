## 🎯 使用指南

### 批量 PDF 导出

**导出步骤：**
1. 点击"工具" → "批量导出 PDF"
2. 或使用快捷键 Ctrl+Shift+P
3. 点击"添加文件"选择多个 Word 文档
4. 设置导出选项（页面大小、质量等）
5. 点击"开始导出"
6. 选择保存目录
7. 等待导出完成

**文件管理：**
- **添加文件：** 点击"添加文件"按钮
- **移除选中：** 选择文件后点击"移除选中"
- **清空列表：** 点击"清空列表"
- **拖拽添加：** 直接拖拽文件到列表

**进度显示：**
- 进度条显示当前导出进度
- 状态文本显示详细日志
- 导出完成后显示统计结果

---

### 中文字体

**自动使用：**
- 软件启动时自动注册中文字体
- PDF 导出时自动使用合适的中文字体
- 无需手动配置

**字体查找路径：**
- Windows: `C:/Windows/Fonts/`
- Linux: `/usr/share/fonts/truetype/`
- macOS: `/Library/Fonts/`

**注意事项：**
- 如果系统未安装中文字体，PDF 可能无法正确显示中文
- 建议安装常见中文字体（宋体、黑体、楷体）

## 📝 使用场景

### 场景 1: 批量导出论文章节

```
1. 完成论文多个章节的格式化
2. Ctrl+Shift+P 打开批量导出
3. 添加所有章节文档
4. 设置 A4 页面大小、高质量
5. 选择保存目录
6. 一键导出所有章节为 PDF
7. 打包发送给导师
```

### 场景 2: 导出多个学生论文

```
1. 收集所有学生的论文文档
2. 批量导出对话框
3. 拖拽所有文档到列表
4. 设置统一导出选项
5. 开始批量导出
6. 自动保存到指定目录
7. 按学生姓名整理 PDF
```

### 场景 3: 中英文混排文档

```
1. 打开包含中文的 Word 文档
2. Ctrl+Alt+P 导出为 PDF
3. 自动使用注册的中文字体
4. 中文和英文都正确显示
5. 无需额外配置
```
