# 🎓 论文排版优化器 v2.0

**专业的学术论文格式化工具**

---

## 🚀 快速开始

### 1. 启动程序

**进入启动文件夹：**
```bash
cd 启动文件
```

**运行程序：**

**Windows:**
```
双击 启动.bat
```

**Linux/Mac:**
```bash
./启动.sh
```

**或命令行：**
```bash
python launcher.py
```

### 2. 安装依赖（首次运行）

```bash
pip install -r requirements.txt
```
