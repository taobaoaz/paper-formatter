## 1. 软件介绍

### 1.1 软件概述

**论文排版优化工具**是一款专为学术论文排版设计的专业软件，支持：

- ✅ 自动识别论文结构
- ✅ 智能格式化处理
- ✅ 多种模板支持（期刊/学位论文）
- ✅ 参考文献自动格式化
- ✅ AI 辅助排版优化

### 1.2 主要功能

| 功能模块 | 说明 |
|----------|------|
| 📄 文档格式化 | 自动调整字体、字号、行距、段落等 |
| 📋 模板管理 | 支持中文期刊、国际期刊、学位论文模板 |
| 🤖 AI 识别 | 自动识别论文结构和格式要求 |
| 📚 参考文献 | 支持 GB/T 7714、APA、MLA、IEEE 等格式 |
| 📊 图表处理 | 自动编号、标题格式化 |
| ✏️ 公式编辑 | 公式编号和对齐 |
| 🔧 批量处理 | 一次处理多个文档 |

### 1.3 系统要求

| 项目 | 要求 |
|------|------|
| **操作系统** | Windows 10/11 (64 位) |
| **Python** | 3.8 或更高版本 |
| **内存** | 至少 4GB RAM |
| **硬盘空间** | 至少 500MB 可用空间 |
| **屏幕分辨率** | 1920×1080 或更高 |

## 2. 快速开始

### 2.1 安装步骤

#### 方法 1：一键启动（推荐）

1. **下载软件包**
   - 从 GitHub 下载完整项目
   - 解压到任意文件夹

2. **双击运行启动脚本**
   ```
   启动程序.bat
   ```

3. **自动安装**
   - 软件会自动检查 Python 环境
   - 自动安装所需依赖包
   - 自动启动程序

#### 方法 2：手动安装

1. **安装 Python**
   - 前往 https://www.python.org/downloads/
   - 下载并安装 Python 3.8+
   - ✅ 勾选 "Add Python to PATH"

2. **安装依赖**
   ```bash
   cd paper-formatter
   pip install -r requirements.txt
   ```

3. **运行程序**
   ```bash
   python main.py
   ```

### 2.2 首次使用

#### 步骤 1：启动软件

双击 `启动程序.bat` 或运行 `python main.py`

#### 步骤 2：选择模板

在主界面选择：
- **中文期刊** - 国内学术期刊
- **国际期刊** - SCI/EI 期刊
- **学位论文** - 本科/硕士/博士论文
- **自定义** - 自定义格式要求

#### 步骤 3：导入论文

点击 **"导入文档"** 按钮，选择你的 Word 文档 (.docx)

#### 步骤 4：格式化

点击 **"开始格式化"** 按钮，等待处理完成

#### 步骤 5：保存结果

点击 **"保存"** 按钮，选择保存位置

## 3. 功能详解

### 3.0 封面和声明页配置

**操作路径：** 工具 → 封面和声明页配置

**功能说明：**
配置论文的封面、独创性声明、版权使用授权书等前置页面。

**配置项：**

#### 封面配置
- ✅ 启用/禁用封面
- 🏫 学校名称
- 📚 论文类型（本科/硕士/博士/课程）
- 🎓 显示校徽
- 📝 封面字段：
  - 论文题目
  - 作者姓名
  - 学号
  - 专业
  - 指导教师
  - 完成日期

#### 声明页配置
- ✅ 启用/禁用声明页
- 📋 声明类型：
  - 独创性声明
  - 版权使用授权书
  - 独创性声明 + 授权书
  - 保密论文声明
- ✍️ 显示签名区域
- 📄 自定义声明内容（可选）

**使用步骤：**
```
1. 点击 "工具" → "封面和声明页配置"
2. 配置封面信息
3. 配置声明页类型
4. 点击 "💾 保存配置"
5. 选择保存位置（.json 文件）
6. 格式化时会自动应用配置
```

**保存配置文件：**
```json
{
  "cover": {
    "enabled": true,
    "school_name": "XX 大学",
    "thesis_type": "本科毕业论文",
    "fields": {
      "论文题目": "基于深度学习的图像识别",
      "作者姓名": "张三",
      "学号": "202012345",
      "专业": "计算机科学与技术",
      "指导教师": "李四 教授",
      "完成日期": "2024 年 6 月"
    }
  },
  "declaration": {
    "enabled": true,
    "type": "独创性声明",
    "show_signature": true
  }
}
```

**加载已有配置：**
```
1. 点击 "📂 加载配置"
2. 选择之前保存的 .json 文件
3. 配置自动填充
```

### 3.1 文档格式化

#### 3.1.1 页面设置

**操作路径：** 设置 → 页面设置

**可配置项：**
- 纸张大小（A4/A3/B5）
- 页边距（上/下/左/右）
- 页眉页脚高度
- 装订线位置

**预设值：**
```
上边距：2.54 cm
下边距：2.54 cm
左边距：3.17 cm
右边距：3.17 cm
```

#### 3.1.2 字体设置

**操作路径：** 设置 → 字体设置

**可配置项：**
- 正文字体（宋体/黑体/楷体等）
- 正文字号（小四/五号等）
- 行距倍数（1.0/1.5/2.0）
- 首行缩进（2 字符）

#### 3.1.3 标题样式

**操作路径：** 设置 → 标题样式

**支持 5 级标题：**
```
一级标题：一、二、三、
二级标题：（一）（二）（三）
三级标题：1. 2. 3.
四级标题：（1）（2）（3）
五级标题：① ② ③
```

**每级标题可设置：**
- 字体
- 字号
- 加粗/斜体
- 对齐方式
- 段前段后间距

## 4. 模板使用

### 4.1 学位论文模板

#### 4.1.1 本科毕业论文

**适用场景：** 本科毕业论文

**格式要求：**
```
纸张：A4
页边距：上 3.5cm，下 2.5cm，左 3cm，右 2.5cm
正文字体：宋体小四
行距：1.5 倍
标题：黑体加粗
```

**使用步骤：**
1. 选择模板类型：学位论文
2. 选择子类型：本科毕业论文
3. 导入论文
4. 应用模板

#### 4.1.2 硕士/博士论文

**特殊要求：**
- 英文摘要必须
- 参考文献不少于规定数量
- 创新性声明

### 4.2 期刊模板

#### 4.2.1 中文期刊

**常见期刊格式：**
- 计算机学报
- 软件学报
- 自动化学报
- 电子学报

**格式特点：**
- 双栏排版
- 摘要 300 字以内
- 关键词 3-5 个
- 参考文献 15 篇以上

#### 4.2.2 国际期刊

**支持格式：**
- IEEE Transactions
- ACM Journals
- Springer
- Elsevier
- Wiley

**格式特点：**
- 英文撰写
- 双栏或单栏
- 参考文献格式多样（IEEE/APA/MLA）

### 4.3 自定义模板

#### 4.3.1 创建自定义模板

1. 点击 **"模板"** → **"新建模板"**
2. 填写模板信息：
   - 模板名称
   - 模板类型
   - 描述说明
3. 配置格式参数
4. 保存模板

#### 4.3.2 模板参数说明

**页面设置：**
- `paper_size`: 纸张大小 (A4/A3/B5)
- `margin_top`: 上边距 (cm)
- `margin_bottom`: 下边距 (cm)
- `margin_left`: 左边距 (cm)
- `margin_right`: 右边距 (cm)

**字体设置：**
- `font_family`: 正文字体
- `font_size`: 正文字号 (磅)
- `line_spacing`: 行距倍数
- `first_line_indent`: 首行缩进 (字符)

**标题设置：**
- `heading1_font`: 一级标题字体
- `heading1_size`: 一级标题字号
- `heading1_bold`: 是否加粗
- `heading1_alignment`: 对齐方式

## 7. 常见问题

### 7.1 安装问题

#### Q1: 双击启动程序没反应？

**可能原因：**
- Python 未安装
- Python 未添加到 PATH
- 依赖包缺失

**解决方法：**
```
1. 检查 Python 是否安装
   - 打开命令提示符
   - 输入：python --version
   - 如果显示版本号，说明已安装

2. 如果未安装 Python
   - 前往 https://www.python.org/downloads/
   - 下载并安装 Python 3.8+
   - ✅ 勾选 "Add Python to PATH"

3. 手动安装依赖
   - cd paper-formatter
   - pip install -r requirements.txt
```

#### Q2: 提示缺少依赖包？

**解决方法：**
```bash
pip install -r requirements.txt
```

**如果安装失败：**
```bash
# 使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 7.2 使用问题

#### Q3: 导入文档失败？

**可能原因：**
- 文件格式不支持
- 文件损坏
- 文件被其他程序占用

**解决方法：**
```
1. 确认文件格式
   - 支持 .docx 和 .tex
   - 不支持 .doc（需另存为.docx）

2. 检查文件是否损坏
   - 用 Word 打开确认

3. 关闭文件
   - 确保文件未被其他程序占用
```

#### Q4: 格式化后格式错乱？

**可能原因：**
- 模板配置错误
- 文档包含特殊格式
- 字体缺失

**解决方法：**
```
1. 检查模板配置
   - 确认模板参数正确

2. 简化文档
   - 删除特殊格式
   - 使用标准字体

3. 安装缺失字体
   - 中文字体：宋体、黑体、楷体
   - 英文字体：Times New Roman、Arial
```

#### Q5: 参考文献格式化不正确？

**可能原因：**
- 原始格式不规范
- 信息缺失
- 格式标准选择错误

**解决方法：**
```
1. 检查原始文献
   - 确保包含完整信息
   - 作者、标题、期刊、年份等

2. 选择正确格式
   - 中文用 GB/T 7714
   - 英文用 APA/IEEE 等

3. 手动调整
   - 使用参考文献编辑器
   - 逐个修正
```

### 7.3 AI 功能问题

#### Q6: AI 识别不准确？

**可能原因：**
- 文档结构复杂
- 格式不规范

2. 手动调整

   - 保存修正结果

3. 反馈问题
   - 记录识别错误
   - 联系开发者改进
```

- 未配置 AI API
- 网络连接问题
- API 密钥无效

1. 配置 AI API
   - 设置 → AI 设置

2. 检查网络
   - 确认网络连接正常

3. 验证 API 密钥
   - 测试 API 连接
```

### 7.4 性能问题

#### Q8: 处理大文件很慢？

**可能原因：**
- 文件过大
- 内存不足
- 电脑性能限制

**解决方法：**
```
1. 分割文件
   - 分章节处理
   - 最后合并

2. 关闭其他程序
   - 释放内存

3. 升级硬件
   - 增加内存
   - 使用 SSD
```

#### Q9: 程序崩溃？

**可能原因：**
- 内存溢出
- 文件损坏
- 程序 bug

**解决方法：**
```
1. 重启程序

2. 检查日志
   - 查看错误日志
   - 定位问题

3. 报告 bug
   - 记录操作步骤
   - 联系开发者

#### Q11: 模板应用后效果不好？

**可能原因：**
- 模板配置错误
- 文档结构不匹配
- 特殊格式冲突

**解决方法：**
```
1. 检查模板配置
   - 确认参数正确

2. 调整文档
   - 简化格式
