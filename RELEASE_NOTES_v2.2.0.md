## 🎯 使用指南

### PDF 导出

**导出步骤：**
1. 打开要导出的文档
2. 点击"工具" → "导出为 PDF"
3. 或使用快捷键 Ctrl+Alt+P
4. 选择导出选项
5. 点击"导出 PDF"

**导出选项：**
- **页面大小：** A4（默认）、A3、B5、Letter、Legal
- **导出质量：**
  - 标准质量：适合屏幕阅读和打印
  - 高质量：适合专业打印
  - 最小文件：适合网络传输
- **压缩：** 启用后减小文件大小
- **嵌入字体：** 推荐启用，确保正确显示

## 📝 使用场景

### 场景 1：导出论文为 PDF

```
1. 完成论文格式化
2. Ctrl+Alt+P 打开 PDF 导出
3. 选择"高质量"
4. 勾选"嵌入字体"
5. 点击"导出 PDF"
6. 发送给导师或投稿
```

### 场景 2：导出多个文档

```
1. 批量格式化多个文档
2. 依次导出为 PDF
```

## 🐛 已知限制

### 当前版本限制

1. **中文字体：** reportlab 默认不支持中文
   - 解决：需要额外配置中文字体
   - 临时方案：使用英文或安装中文字体

2. **复杂格式：** 某些 Word 格式可能无法完美转换
   - 解决：持续优化转换算法

3. **批量导出：** 暂不支持一键批量导出
   - 解决：未来实现批量导出功能

### 未来改进方向

1. **中文字体支持：** 内置中文字体
2. **格式保真：** 提高格式转换保真度
3. **批量导出：** 一键导出多个文档
4. **云存储：** 直接导出到云盘
