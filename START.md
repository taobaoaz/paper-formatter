# 🚀 启动程序

## 快速启动

### Windows 用户
**双击运行：** `启动.bat`

### Linux/Mac 用户
**终端运行：** `./启动.sh`

### 所有平台
**命令行：** `python launcher.py`

---

## 📋 启动文件

| 文件 | 说明 | 平台 |
|------|------|------|
| `launcher.py` | Python 启动入口 | 所有 |
| `启动.bat` | Windows 快捷启动 | Windows |
| `启动.sh` | Linux/Mac 快捷启动 | Linux/Mac |

---

## ⚙️ 首次使用

### 安装依赖
```bash
pip install -r requirements.txt
```

### 启动
```bash
# Windows
启动.bat

# Linux/Mac
./启动.sh

# 或
python launcher.py
```

---

## ✨ 新功能

启动后，新功能在：
- **工具栏：** 📘 封面配置
- **菜单：** 工具 → 封面和声明页配置
- **快捷键：** Ctrl+Shift+C
