## 🎯 使用指南

### 批量处理

**步骤：**
1. 工具 → 批量处理（或 Ctrl+B）
2. 添加要处理的文件
3. 勾选"自动备份"
4. 点击"开始处理"
5. 查看处理日志

### 自动备份

**默认行为：**
- ✅ 格式化前自动备份
- ✅ 备份文件保存在 `backup/` 目录
- ✅ 自动保留最近 3 个备份

**备份文件名格式：**
```
原文件名_backup_YYYYMMDD_HHMMSS.docx
```
